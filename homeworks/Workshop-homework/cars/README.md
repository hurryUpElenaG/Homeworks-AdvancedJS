# HOMEWORK

Full list of homework for the course **Advanced JavaScript and AI**. Each homework is divided into 3 parts: `Basic`, `Medium`, and `Advanced`.

> **AI Assistance Tip:** You can use AI (like ChatGPT or GitHub Copilot) to get help with logic, code examples, and debugging. Always read and understand the AI suggestions before using them in your project.

---

## Homework 1

### Basic

**Task:**
- Place the JSON `cars.json` in your GitHub repository.
- Use the `fetch` method to get the data from the server.
- Display the cars data in a table.

**AI Guidance & Prompts 🤖:**
- "Generate JavaScript code to fetch a JSON file named cars.json and display it in an HTML table."
- "How do I dynamically create table rows from an array of objects in JavaScript?"
- "Write a function that logs each car object from a JSON file to the console before displaying it in the table."

---

### Medium

**Task:**
- Add inputs to filter the data:
  - By type (select)
  - By brand (select)
- Add a search button. When clicked, the table should filter the cars based on the selected type and brand.

**AI Guidance & Prompts 🤖:**
- "Show me a JavaScript function that filters an array of car objects by selected type and brand from HTML select inputs."
- "How can I update an HTML table after filtering JSON data dynamically?"
- "Write an example where selecting options in a form filters a table in real time using JavaScript."

---

### Advanced

**Task:**
- Add additional filters:
  - By model (text input)
  - By doors (number input)
  - By gasType (select)
  - By color (text or select)
  - By new/old (radio buttons or checkbox)
  - By horsepower (number or range input)
- Ensure all filters work together. Example: selecting brand BMW and color white should show only BMW cars that are white.

**AI Guidance & Prompts 🤖:**
- "Write a JavaScript function that applies multiple filters to a JSON array at once using HTML input values."
- "Implement a search that matches partial strings, e.g., typing 'ol' matches 'Polo' and 'Volvo'."
- "Show how to combine select, text, number, and radio inputs to filter JSON data in a table."

---

### Bonus

1. Add a button `Reset filters` to reset all filters and restore the table.
2. Display `No data found` if no results match the filters.
3. Implement partial searching for text inputs (e.g., typing 'ol' matches 'Polo' and 'Volvo').

**AI Guidance & Prompts 🤖:**
- "Write a JavaScript function that resets all input fields and restores the original table data."
- "Show how to display a message 'No data found' if a filtered array is empty."
- "Write code that performs case-insensitive partial string matching on an array of objects."

---

### Extra AI Tips

- Use AI to suggest more efficient filtering logic.
